# Simple-Quiz-Application
📚 Quiz Application
🔹 Overview
A simple and interactive Quiz Application
Built using HTML, CSS, and JavaScript
Allows users to answer multiple-choice questions and view their score

🔹 Features
Multiple-choice questions
Instant answer selection
Score calculation at the end
Responsive design (works on mobile & desktop)
Clean and user-friendly interface

🔹 Technologies Used
HTML (Structure)
CSS (Styling)
JavaScript (Logic & Functionality)

🔹 How It Works
User starts the quiz
Questions are displayed one by one
User selects an option
Answers are checked using JavaScript
Final score is shown at the end

🔹 Project Structure
index.html → Main structure of the app
style.css → Styling and layout
script.js (or inside HTML) → Quiz logic and functionality

🔹 Learning Outcomes
DOM Manipulation
Event Handling
Basic JavaScript Logic
Building interactive UI

🔹 Future Improvements
Add timer for each question
Add more questions dynamically
Store high scores
Add categories/difficulty levels
